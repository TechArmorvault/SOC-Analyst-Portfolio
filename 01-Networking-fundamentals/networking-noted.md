# Networking Fundamentals

## What is an IP Address?

An IP address is a unique address assigned to a device on a network that allows communication between devices.

## TCP vs UDP

TCP is connection-oriented and ensures reliable delivery.
UDP is faster but does not guarantee delivery.

## Common Ports

- 22 SSH
- 53 DNS
- 80 HTTP
- 443 HTTPS
- 3389 RDP

## OSI Model

The OSI model has 7 layers:
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application
